# GoMelb

This project focuses on implementing the SQL logic for the GoMelb application. It includes the database schema (DDL), indexes, views, materialized views, functions, procedures, triggers, and advanced SQL queries used to support the application's functionality and business logic.

The complete project documentation, including the application overview, requirements, and implementation details, is available in the original GoMelb project page:

https://develop.finki.ukim.mk/projects/GoMelb
